🧠 Emotion-Aware User Interface System

An Adaptive Exam Interface Based on Student Stress that dynamically adjusts the user experience during online examinations by detecting and responding to student stress levels in real time.

📌 Overview

With the rise of online examinations, students often face stress and anxiety that negatively affect performance. Traditional systems use a static interface, ignoring individual emotional states.

This project introduces an emotion-aware adaptive system that:

Monitors student behavior
Detects stress levels
Dynamically adjusts the exam interface
🎯 Objectives
Detect student stress using behavioral patterns
Adapt UI in real-time for better user experience
Reduce cognitive load and anxiety during exams
Improve fairness and performance evaluation
🚀 Key Features
🎯 Real-Time Stress Detection
Based on response time, typing speed, mouse movement, etc.
🎨 Adaptive User Interface
Changes colors, layout, and difficulty dynamically
💬 Motivational Prompts
Encouraging messages during high stress
⏱️ Flexible Time Adjustment
Extra time support when needed
📊 Post-Exam Feedback
Performance insights and stress analysis report
